== Description ==

Dashboard Links Terminator v2 Removes the  WordPress logo and drop down menu with list of links from the top toolbar/admin bar so that visiting users would not be confused on the site they are on. Brings consistency.
 
Easy to use with user friendly control panel

DLT v2 has the ability to:

1. Replace logo with a text

2. Replace logo with a text and link

3. Replace logo with your own image

4. Replace logo with your own image and link

5. Change text color - also a option to change the mouse hover over effect text color*

6. Add a link (leave it blank if you like) - or add a link and change the hover over effect of that link!


Your WordPress toolbar owns your site now!

With the plugin you can do alot, make your website look more effective and original in the toolbar!

Have any questions?
Ask away on our support <a href="http://www.trottyzone.com/forums/forum/wordpress-plugins/">forum</a>
== Installation ==

1. Upload the entire 'Dashboard Links Terminator' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy! ;)

== Frequently Asked Questions ==

= How do I see changes after update? =

After changes are saved, the effects are not instant, you would have to revert to your site or click on any other link (navigate away) for the changes to take effect.


= How do I deactivate the WordPress Logo totally? =

Just tick the checkbox right above the "Save changes" buttton


= How do i use this plugin? =

Simple just follow the instructions for installation and remember to activate and thats it!
